EmailObfuscator 1.0.0

A custom module for ProcessWire2.1+

Original version by 12345j
Modified by Philipp "Soma" Urlich, philipp at urlich dot ch

http://processwire.com/talk/topic/497-email-obfuscator-module

This Module will add an alert message for pages already being edited by another user. It will show name and email. Page will still be editable to your own risk.

How to install:
- Download the contents of this repository and put the folder renamed as "EmailObfuscator" into your site/modules/ folder
- Login to processwire and got to Modules page and click "Check for new modules". You should see a note that two new modules were found. Install the module.
- Done
